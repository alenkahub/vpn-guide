# Гайд на фришный ВПНчик от злых ручек РКНа.

Первое, что мы делаем - переходим на зеркало Амнезии: https://m-d32971ff1d7d-3w5hsuiikq-ez.a.run.app/ru/downloads и качаем клиент приложения, либо с гитхаба: https://github.com/amnezia-vpn/amnezia-client/releases/download/4.8.15.4/AmneziaVPN_4.8.15.4_x64.exe

Далее - устанавливаем, у нас открывается окошко с настройками подключения, наш вариант - файл с настройками подключения. Переходим на сайтик: https://warp-generator.github.io/ и генерируем конфиг, любой из вариантов в целом подойдет, а на счет стабильности - экспериментируйте. Импортируйте файлик в амнезию, подключаетесь и готово :)
